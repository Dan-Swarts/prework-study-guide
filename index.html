HTML

  • Each HTML page is comprised of elements.

  • An element comprises a pair of tags, and content between the tags

  • An element can also have attributes inside the opening tag


  • The tag defines which kind of element. There are many different tags

  • Different kinds of elements:

    Main elements: these are present in every HTML page, and are the most basic/most important

    • The <html> element is the root element: the stating point for every HTML page.

    • The <head> element contains information about the web page that the user doesn't see

    • The <body> element contains all content that the user will see on the page

    Content elements: These appear in the body, and show all the content visible to the user

    • The <h1>,<h2>,<h3>,<h4>,<h5>, and <h6> tags stand for "header 1" through "header 6." Header 1 is biggest, and 6 is smallest.

    • The <p> tag stands for "paragraph", and represents some text

    • The <img> tag stands for "image," Which displays a picture on the page

    • The <ul> tag stands for "unordered list," which is a bulleted list.

    • The <ol> tag stands for "ordered list," which is a numbered list.

    • The <li> tag stands for "list item," and is one entry in the list

    Semantic elements: these don't present content, but show the structure of the page

    • <section>
